# Brand-System KIDS e.V.

Das Brand-System des KIDS e.V. ist verspielt, kindlich, farbenfroh und warm - ohne kitschig zu werden. Entworfen von **zartmint.de**.

## Kernbausteine

| Baustein | Datei | Kurzbeschreibung |
| --- | --- | --- |
| Farben | [colors.md](colors.md) | 7 Primärfarben + Sandbraun als neutraler Ton |
| Logos | [logos.md](logos.md) | Primär-, Sekundär-, rund, schwarz, weiß |
| Typografie | [typography.md](typography.md) | Maybug MS (4 Schnitte) |
| Deko / Störer | [decorative.md](decorative.md) | 11 einzelne Deko-Elemente |

## Tonalität

- **Freundlich, nahbar** - wir sprechen Eltern auf Augenhöhe an.
- **Spielerisch, aber nicht albern** - ansprechend für Erwachsene, warm für Familien.
- **Ehrlich und konkret** - keine leeren Marketingphrasen, sondern was wir wirklich tun.
- **Inklusiv** - gender-sensible Sprache (z. B. „Eltern\*innen" nur wenn konsistent; sonst „Eltern", „Familien", „Kinder").

## Einsatzprinzipien

1. **Großzügiger Weißraum.** Die Marke lebt vom Atmen-Können der Deko-Elemente. Nicht überladen.
2. **Deko-Elemente als Akzente**, nicht als Muster. Sie kleben nicht am Logo, sondern streuen um Inhalte herum.
3. **Maximal 3 Farben pro Komposition** (plus Weiß + ein Textton). Alle 7 Farben gleichzeitig erzeugt visuelles Chaos.
4. **Regenwurm (Mascot) sparsam einsetzen.** Er ist Charakter, nicht Deko. Gut auf Startseite, Fehlerseiten, Erfolgsmeldungen.
5. **Maybug MS nur für Headlines und Akzente.** Für Fließtext eine gut lesbare Web-Schrift (siehe [typography.md](typography.md)).
