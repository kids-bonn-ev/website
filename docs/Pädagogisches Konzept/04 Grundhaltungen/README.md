# Grundhaltungen

In unserer pädagogischen Arbeit legen wir besonderen Wert auf die Förderung von **Selbstständigkeit**, **Partizipation**, **Selbstbestimmung** und **Wertschätzung**. Diese Grundhaltungen prägen unseren Alltag und unser Miteinander und bilden die Basis für eine kindgerechte, demokratische und respektvolle Bildungseinrichtung.
