# Teiloffene Arbeit

Die teiloffene Arbeit stellt eine pädagogische Organisationsform dar, die Elemente fester Gruppenstrukturen mit offenen Bildungsangeboten verbindet. Sie vereint die Vorteile eines sicheren Rahmens mit den Möglichkeiten individueller Entfaltung und Selbstbestimmung. Im KIDS bildet die teiloffene Arbeit eine zentrale Grundlage unseres pädagogischen Handelns.
